# Bro ransomware
Bro policy for ransomware detection over SMB2.

## Entropy
By calculating the entropy of the data that is being send in the first write-request, the policy is able to determine if a file has been encrypted.

## Detected Ransomware
- Locky
- Cryptowall
- CTBLocker
- Jigsaw
- Mobef
- Shade
- Maktub
- Cerber/Alpha
- Teslacrypt
- Rokku
